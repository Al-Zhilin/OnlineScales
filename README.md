## *Проект электронных весов с удаленным мониторингом показаний на ESP32-S3*

### *Краткий обзор:*

- Использование ESP32-S3 N16R8 в качестве основного МК
- Компенсация температурного дрейфа тензодатчиков с помощью термометра DS18b20 и подбора лучшей коменсационной RLS модели в процессе калибровки
- Интеграция с Yandex IoT Core и Yandex DataLens для KPI мониторинга показателей
- Поддержка выхода в сеть через SIM800L (мобильная сеть через SIM-карту) или через WiFi (разрабатывается)
- Адаптивные модели поведения при различных неисправностях
- "Проксирование" запросов данных через VK Callback API для нормальной работы при введении режима "белых списков"
